## 1. Brute Force

::tabs-start

```python
class Solution:
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        s.sort()
        res = 0

        for i in g:
            minIdx = -1
            for j in range(len(s)):
                if s[j] < i:
                    continue
                
                if minIdx == -1 or s[minIdx] > s[j]:
                    minIdx = j
            
            if minIdx != -1:
                s[minIdx] = -1
                res += 1
        
        return res
```

```javascript
     * @param {number[]} g
     * @param {number[]} s
     * @return {number}
     */
    findContentChildren(g, s) {
        s.sort((a, b) => a - b);
        let res = 0;

        for (let i of g) {
            let minIdx = -1;
            for (let j = 0; j < s.length; j++) {
                if (s[j] < i) continue;

                if (minIdx === -1 || s[minIdx] > s[j]) {
                    minIdx = j;
                }
            }

            if (minIdx !== -1) {
                s[minIdx] = -1;
                res++;
            }
        }

        return res;
    }
}
```

### Time & Space Complexity

* Time complexity: $O(n * m + m \log m)$
* Space complexity: $O(1)$ or $O(m)$ depending on the sorting algorithm.

> Where $n$ is the size of the array $g$ and $m$ is the size of the array $s$.

## 2. Two Pointers - I

::tabs-start

```python
class Solution:
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        g.sort()
        s.sort()
        
        i = j = 0
        while i < len(g):
            while j < len(s) and g[i] > s[j]:
                j += 1
            if j == len(s):
                break
            i += 1
            j += 1
        return i
```

### Time & Space Complexity

```python
class Solution:
```

```java
public class Solution {
```
